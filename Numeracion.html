Viñetas y Numeracion

Numeracion

En la numeracion se utiliza la instruccion <ol> </ol> Cada uno de los elemntos de la lista esta predecido de la instruccion <li>. Estas en la instruccion <ol> se puede incluir el tipo numeracion con el atributo "Type". Ejemplo:

En codigo En pagina
    <ol>
            <li>Elemento 1</li>
            <li>Elemento 2</li>
            <li>Elemento 3</li>
    </ol>
  1. Elemento 1
  2. Elemento 2
  3. Elemento 3

Type=indica el tipo de numeracion utilizada. Si no se indica se entiende que sera una lista ordenada numericamente. Los tipos posibles son:

Type=... Ejemplo
Typer="1"
  1. Elemento 1
  2. Elemento 2
  3. Elemento 3
type="a"
  1. Elemento 1
  2. Elemento 2
  3. Elemento 3
type="A"
  1. Elemento 1
  2. Elemento 2
  3. Elemento 3
type="i"
  1. Elemento 1
  2. Elemento 2
  3. Elemento 3
type="I"
  1. Elemento 1
  2. Elemento 2
  3. Elemento 3

Viñetas

Las viñetas se representan con la instruccion <ul> </ul> para delimitar la lista, y <li> para indicar cada uno de los elementos. En la instruccion <ul>.

En codigo En pagina
    <ul>
            <li>Elemento 1</li>
            <li>Elemento 2</li>
            <li>Elemento 3</li>
    </ul>
  • Elemento 1
  • Elemento 2
  • Elemento 3

Type=indica el tipo de viñeta utilizada. Si no se indica se entiende que sera una viñeta tipo disk. Los tipos posibles son:

Type=... Ejemplo
Typer="disk"
  • Elemento 1
  • Elemento 2
  • Elemento 3
type="circle"
  • Elemento 1
  • Elemento 2
  • Elemento 3
type="Square"
  • Elemento 1
  • Elemento 2
  • Elemento 3

Listas Anidadas

Puedes anidar listas para representar subelementos de un elemento de lista.

En codigo En pagina
    <ul>
            <li>item 1</li>
            <li>item 2
                    <ul>
                            <li>sub-item 2.1</li>
                            <li>sub-item 2.2</li>
                    </ul>
            </li>
            <li>item 3</li>
    </ul>
  • item 1
  • item 2
    • sub-item 2.1
    • sub-item 2.2
  • item 3
